# My Cypress Automation

This project utilizes Cypress to automate the testing of a login feature and user update feature.
Cypress is a JavaScript-based end-to-end testing framework that allows depelopers to write tests and execute
them in a real browser.

(PT-BR) Este projeto utiliza Cypress para automatizar o teste de um recurso de login e de atualização do usuário.
Cypress é uma estrutura de teste ponta a ponta baseada em JavaScript que permite que os depelopers escrevam testes e executem
-los em um navegador real.

## Installation
```bash
npm install
```
> **NOTE**
> 
> it is necessary to have **node** installed

## Running
```bash
# You open the Cypress UI
npx cypress open

# Run the automation by command line
npx cypress run
```
